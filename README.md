# Allane Leasing UI, Allane Leasing BE and Allane Leasing DB

## Technologies used - front-end:
* Angular
* Angular Material
* OpenApi Generator
* Docker
* Tilt - https://tilt.dev/

## Technologies used - back-end:
* Spring Boot
* Gradle
* Project Lombok
* OpenApi Generator
* MariaDB
* Flyway
* Docker
* Kubernetes
* Tilt - https://tilt.dev/

## Run (Recommended)

Run all apps (front-end and back-end) using the tilt file from root directory.

```
tilt up
```
Stop the app

```
tilt down
```
